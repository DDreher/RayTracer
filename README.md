# VulkanSandbox

A simple path tracer written in C++.

## How to build (Win64)

1. Run `GenerateProjectFiles.bat`.
2. Open the generated solution with VS2019.
3. Build and run in the desired configuration (debug / release)

## Dependencies

* [STB](https://github.com/nothings/stb)
